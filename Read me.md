# RebateWire Oklahoma – HVAC Rebate MVP (March 2026)

Production-ready Next.js 14 SaaS for Oklahoma contractors.

## Quick Start
1. `npm create next-app@latest rebate-wire-ok --typescript --tailwind --eslint --app`
2. Replace all files with the code below
3. `npm install @supabase/supabase-js zod react-hook-form lucide-react @react-pdf/renderer`
4. Create Supabase project → copy URL + keys into .env.local
5. Paste schema into Supabase SQL editor
6. `npm run dev`
7. Click “Seed Oklahoma Demo Data” on empty dashboard
